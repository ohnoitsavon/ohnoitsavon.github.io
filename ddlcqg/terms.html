Privacy Policy

In order for the DDLCQG Doi Bots to function some data is required to be stored. The following is a list of the data we keep along with why and how we used the data:

The Guild/Server ID
This is used as an ID for the bot settings, so we know which server the settings apply to.

Settings/Options selected using the /settings command
Any options selected are saved and linked to the server ID so that the selected settings can be saved and used by the bot when other commands are run in that respective server.

Deleted User IDs/IDs of users that are timed out
When a user is deleted or otherwise timed out from using bot commands, the user ID is saved. The bot can then check against this list preventing the user from using the bot. When the user is restored, the timeout is complete, or the bot is restarted the user IDs are deleted.

User IDs of solo poem games
When a user starts a solo poem game, their user ID is saved. This allows the bot to check which user is playing the game. Once the game is finished or the bot is restarted, the user IDs are deleted.

This data is not shared with anyone other than Discord and the bot developers.

If you have any concerns about the bot or wish to have your data removed please contact the bot developers on the support server, or by emailing ohnoitsAvon@gmail.com.
